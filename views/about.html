About This Project

Movie Library is a full-stack web application developed as part of university backend assignments.

The project demonstrates how a real movie catalog platform can be built using Node.js, Express.js, and MongoDB, with full CRUD functionality available directly through the website.

Users can browse movies, filter them by year, and interact with the catalog through a modern dark-themed interface inspired by popular streaming platforms.

Key Features

  • RESTful API with MongoDB
  • Full CRUD operations via web interface
  • Filtering and sorting support
  • Deployment-ready configuration with environment variables
  • Modern dark UI design

This project serves as a foundation for further extensions such as authentication, admin roles, and user-specific features.
